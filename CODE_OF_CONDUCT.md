# Code of conduct

Contributors must communicate respectfully, assume good intent while addressing concrete behavior,
and welcome people regardless of identity, background, ability, or experience. Harassment,
discrimination, threats, sexualized conduct, doxxing, and sustained disruption are unacceptable.

Report conduct concerns privately to the repository owner. Reports will be handled confidentially,
with proportionate actions ranging from a warning to permanent exclusion. This policy applies in
repository spaces and when representing the project elsewhere.
