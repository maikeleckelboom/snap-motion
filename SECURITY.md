# Security policy

Snap Motion is private beta software and has no supported public release yet.

Report suspected vulnerabilities privately through GitHub's security-advisory flow. Do not open a
public issue containing exploit details, credentials, private repository content, or personal data.
Include affected versions, a minimal reproduction, impact, and any known mitigations. A maintainer
will acknowledge a complete report within seven days and coordinate disclosure after a fix exists.

Only the latest beta candidate is eligible for security fixes. Physical accessibility testing and
package-name ownership remain release blockers; they are not security certifications.
